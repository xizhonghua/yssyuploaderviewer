YSSY Upload Viewer
==================
A Chrome Extension for YSSY BBS (http://bbs.sjtu.edu.cn)

Features
--------

*   Auto load images on the upload page
*   Auto resize images to fit the window
*   Hide irrelevant items
*	Auto load next/previous page
*	Shortcuts supported
*	Progress bar

Shortcuts
---------
*   J : Next image
*   K : Previous image
*   L : Next page
*   H : Previous page
*   G : Goto item
*   B : Goto board
